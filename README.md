# Youtube 클론코딩

[유튜브 오픈 API](https://developers.google.com/youtube/v3/docs)를 사용하여 실제 유튜브 데이터를 이용한 클론코딩


## 💻 사용 기술

- `css` : Tailwind CSS
- `React`
- **Libraries** : react-router-dom, react-query, axios, react-icons, timeago.js

## 📅 프로젝트 관리

- git
- GitHub
  
## 📌 주요 기능
**헤더**
- 키워드로 검색 가능

**전체 비디오 목록**
- 트렌디한 비디오 목록으로 출력

**비디오 상세 페이지**
- 우측에 연관된 비디오 목록 출력

**라우팅 적용**
- 검색하고 클릭한 것들 뒤로가기, 앞으로 가기 가능

**스타일링**
- 반응형 적용
